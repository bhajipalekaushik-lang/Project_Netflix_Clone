# Project_Netflix_Clone

# Project Overview:
This project is a Netflix Homepage Clone built using HTML and CSS.
The goal of this project is to practice frontend development skills, improve layout designing, and understand how modern streaming platform interfaces are structured.
It recreates the look and feel of Netflix’s landing page, including navigation bar, hero section, background image styling, buttons, and responsive layout structure.

# Features
- Netflix-style homepage layout
- Modern UI design using pure CSS
- Responsive structure (basic responsiveness)
- Navigation bar with logo and buttons
- Hero section with background image
- Styled buttons (Sign In / Get Started)
- Section-based content layout

# Technologies Used
HTML5
CSS3
Flexbox
Background styling
Positioning techniques
